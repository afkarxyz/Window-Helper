## Resize The Window

- Win + 1 = 480p 1:1
- Win + 2 = 480p 16:9
- Win + 3 = 480p 21:9
- Win + 4 = 720p 1:1
- Win + 5 = 720p 16:9
- Win + 6 = 720p 21:9
- Win + 7 = 1080p 1:1
- Win + 8 = 1080p 16:9
- Win + 9 = 1080p 21:9
- Win + 0 = Windowed / Fullscreen

## Change Window Position

- Win + ⬆️ = Top
- Win + ⬇️ = Down
- Win + ⬅️ = Left
- Win + ➡️ = Right
- Win + C = Center

## Reposition The Window

- Win + Numpad 7 = ↖️
- Win + Numpad 8 = ⬆️
- Win + Numpad 9 = ↗️
- Win + Numpad 4 = ⬅️
- Win + Numpad 5 = Center
- Win + Numpad 6 = ➡️
- Win + Numpad 1 = ↙️
- Win + Numpad 2 = ⬇️
- Win + Numpad 3 = ↘️
